The importance of Semantic Tags

Semantic HTML 5 tags provide meaning to the web content. These tags help search engines and other user deviced to determine the significance and context of web pages

Using semantic tags, we make our webpage more accessible and improve SEO performance

Key HTML 5 Tags

The following are some of the key HTML5 semantic tags:

  • article :
    Represent a self-contained piece of content
  • section
    Defines a section within a document
  • aside
    Represent content related to the main content, often used for sidebars
  • figure
  • figcaption

Sidebar

This is an aside section where related content such as links to related articles or advertisement, might appear
